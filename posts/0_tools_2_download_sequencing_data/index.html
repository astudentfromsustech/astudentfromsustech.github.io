Download the public sequencing data using kingfisher

March 17, 2025

  • tool
  • kingfisher
  • data

Kingfisher is quite convinient for us to download publica sequencing data

1. Install kingfisher

conda install -c conda-forge -c bioconda kingfisher # pay attention to the channel order

2. Examples of downloading date using kingfisher

kingfisher get -p PRJNA938179 -m aws-http prefetch aws-cp gcp-cp ena-ftp -f fastq --check-md5sums
kingfisher get -p PRJNA938179 -m aws-http prefetch aws-cp gcp-cp ena-ftp -t 16 -f fastq.gz --check-md5sums
kingfisher get -r SRR12044307 -m aws-http prefetch aws-cp gcp-cp ena-ftp -f fastq --check-md5sums
kingfisher get -r SRR12044307 -m aws-http prefetch aws-cp gcp-cp ena-ftp -t 16 -f fastq.gz --check-md5sums

kingfisher get --run-identifiers-list srr_list.txt -m aws-http prefetch aws-cp gcp-cp ena-ftp -t 16 -f fastq.gz --check-md5sums
