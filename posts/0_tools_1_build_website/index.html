#Build personal website and host it on github

1. Using Hugo with Gokarna as the theme

a. Follow the Gokarna tutorial to install GO, HUGO, and git

b. Create the project and install Gokarna theme

hugo new site my-website
cd my-website
git init
git submodule add https://github.com/gokarna-theme/gokarna-hugo.git themes/gokarna

d. Launching the Website Locally

hugo server

2. Host the public diretory on github using a seperate branch

a. Generate the SSH Key

ssh-keygen -t ed25519 -C "your_email@example.com

b. Add your SSH key to the agent

# for Ed25519 key, if you’re using RSA, use the corresponding file (e.g., ~/.ssh/id_rsa). 
ssh-add ~/.ssh/id_ed25519

c. Add your public key to GitHub

# copy the content of ~/.ssh/id_ed25519.pub or ~/.ssh/id_rsa.pub
pbcopy < ~/.ssh/id_ed25519.pub # or  pbcopy < ~/.ssh/id_rsa.pub
# paste the public key to the “Key” field (click the New SSH key in GitHub SSH Settings)

d. Keep public in a separate branch (Another choice is to directly push the public directory)

# Remove the .git Folder in public
cd /Users/fayevalentine/Desktop/tools/tools_2_hugo/hugo/ypf_website/public
rm -rf .git 

# Clone Your GitHub Pages Repository Locally
cd ~/repos
git clone git@github.com:astudentfromsustech/astudentfromsustech.github.io.git  

# Copy the generated public directory to the repository root
cd astudentfromsustech.github.io
cp -R /Users/fayevalentine/Desktop/tools/tools_2_hugo/hugo/ypf_website/public/* .  

# Commit and Push the Files
git add .
git commit -m "Publish Hugo site: update GitHub Pages site"
git push -u origin master
