HUGO, follow-up

March 2, 2024

  • HUGO

Hello, HUGO!

Why?

Inspired by the website of Prof.XiChen, I also want to build my own website using HUGO with gokarna theme. I think it’s a good way to record my thoughts and progresses, as well as a method to present myself. Besides, it can also encourage me to write more, which I think, is quite important to reorganize various pieces of knowledge and assimilate them into my own understanding.

How?

First, install HUGO and download the theme called gokarna. Then I learn the basic rules of building a website using HUGO(Referenc Video)

Second, I choose the StackEdit/PyCharm as the MarkDown editor and quickly learn some basis of MarkDown syntax (Reference Video)

Last, I use github to publish website: https://astudentfromsustech.github.io/

Troubleshooting

At first, I thought I had successfully built the website, but there were many mistakes:

  1. Some links were added to the posts, but they did not redirect as I expected.
  2. Images (including the avatar and the figures inserted in the post) didn’t display.
  3. My content and modifications did not update on github

I realized that the key problems mainly came from two aspects:

  1. I am not familiar with HUGO (or javascipt), so I do not know which path should the images be put.
  2. I am not familiar with the basis of github.

ChatGPT can help me quickly know the basis of github but helps little in javascript because I know too little about javascript to interact with ChatGPT. After I tried 4 or 5 times, I solved these problems. Besides, I write an update_scipt.sh so that I can update easily. And this blog is also written to test whether the script works as I expect.
