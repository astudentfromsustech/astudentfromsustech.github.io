Computer hardware

Key components to assemble a computer and their functions:

1. Motherboard

The motherboard is the main printed circuit board (PCB) in a computer. It is the central communication backbone connectivity point through which all components and external peripherals connect.

Functions:

  1. Component Integration: The motherboard houses the CPU, memory (RAM), and provides connectors for other peripherals like the graphics card, hard drives, and optical drives.
  2. Connectivity: It offers various connectivity options including SATA (for hard drives and SSDs), USB ports, Ethernet, audio jacks, and sometimes integrated Wi-Fi.
  3. BIOS/UEFI Firmware: The motherboard contains a chip with firmware (BIOS or UEFI) that manages the hardware initialization during the boot-up process and provides runtime services for operating systems and programs.
  4. Data and Power Distribution: It distributes power to various components and facilitates the transfer of data between them through buses and circuits embedded on the board.

2. CPU

Intel CPU Series
  1. Intel Core Series: Core i3, Core i5, Core i7, Core i9 (for gamer)
  2. Intel Xeon: Designed for servers and workstations, Xeon processors offer stability, scalability, and are optimized for multi-threaded applications and heavy workloads.
  3. Intel Pentium and Celeron: These are budget-friendly options suitable for basic computing needs, like web browsing, document editing, and other light tasks.
  4. Intel Atom: Used mainly in compact and low-power devices like tablets and low-cost laptops. They are optimized for energy efficiency rather than performance.
AMD CPU Series
  1. AMD Ryzen Series: Ryzen 3: Comparable to Intel’s Core i3, Ryzen 9: Enthusiast-level, akin to Intel’s Core i9.
  2. AMD Threadripper: Targeted at the high-end desktop market, these CPUs offer even more cores and threads than the Ryzen 9 series, making them ideal for professional content creators, designers, and heavy multitaskers.
  3. AMD Athlon: Budget-oriented CPUs that are great for everyday computing tasks, positioned similarly to Intel’s Pentium and Celeron.
  4. AMD EPYC: Designed for servers, AMD EPYC processors offer high core counts and are built to handle intense workloads, competitive with Intel’s Xeon.

3. Random Access Memory (RAM)

RAM modules are installed in RAM slots on the motherboard. These slots are typically near the CPU and are standardized for easy access and upgradeability. The form factor for desktop RAM is usually DIMM (Dual In-line Memory Module).

Manufacturers: Kingston Technology, Samsung Electronics

4. Storage

Storage devices are used to store data permanently. This includes hard disk drives (HDDs), solid-state drives (SSDs), and sometimes older mediums like optical drives (CD/DVD). SSDs are faster than HDDs and are becoming more common in modern computers.

Manufacturers:

Western Digital (WD):
a) WD Blue, Black, and Red HDDs: Target different uses, such as general computing, gaming/performance, and network attached storage (NAS) systems, respectively. b) WD Green SSDs: Focus on energy efficiency with lower performance than other SSDs, suitable for basic use. c) WD Black SSDs: High-performance SSDs designed for gaming and high-intensity applications.

5. Graphics Processing Unit (GPU)

While many CPUs have integrated graphics, a separate GPU (or graphics card) is used in systems where high graphic performance is needed, such as in gaming or graphic design. The GPU accelerates the creation of images, videos, and animations.

A graphics card, also known as a video card or display card, is an expansion card which generates a feed of output images to a display device (like monitors). Typically, a graphics card is a printed circuit board (PCB) that houses the GPU along with the following components:

a. GPU (Graphics Processing Unit): The heart of the graphics card, responsible for processing images and video data. b. Output Connectors: These are interfaces like HDMI, DisplayPort, DVI, or VGA through which the graphics card connects to display devices. c. VRAM (Video RAM): Dedicated memory used by the GPU to store image data that will be displayed. This allows the GPU to access this information quickly without straining the system’s main RAM.

6. Output Devices: Output devices display, project, or physically print information from

the computer, such as monitors, projectors, and printers.

7. Input Devices: These include tools used to enter information into the computer,

such as keyboards, mice, scanners, and webcams.

HDMI (High-Definition Multimedia Interface) Function: Transmits both high-definition video and audio from the computer to the monitor.

8. Networking Components: These components, either integrated into the

motherboard or added separately, allow the computer to connect to a network. They can include Ethernet ports and Wi-Fi adapters.

9. Power Supply Unit (PSU): The PSU converts the AC (alternating current)

from a wall outlet into DC (direct current) used by the computer’s internal components.

10. Cooling System: Computers generate heat, especially the CPU and GPU.

Cooling systems, including fans and heat sinks, are crucial to dissipate heat and keep the computer running efficiently. More advanced systems might use liquid cooling.
