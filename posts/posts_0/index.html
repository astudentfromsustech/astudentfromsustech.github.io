Hello, HUGO!

Why?

Inspired by the website of Prof. Xi Chen, I also want to build my own website using HUGO with gokarna. I think it’s a good way to record my thoughts and progresses, as well as a method to present myself. Besides, it can also encourage me to write more, which I think, is quite important to reorganize various pieces of knowledge and assimilate them into my own understanding.

How?

First, install HUGO and download the theme called gokarna. Then I learn the basic rules of building a website using HUGO(Referenc Video)

Second, I choose the StackEdit/PyCharm as the MarkDown editor and quickly learn some basis of MarkDown syntax (Reference Video)

Last, I use github to publish website: https://astudentfromsustech.github.io/
